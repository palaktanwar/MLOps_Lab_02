# Docker Lab 1 - Breast Cancer KNN Model

### Build the Docker image:
```bash
docker build -t lab1:v1 .
```

### Run the container:
```bash
docker run lab1:v1
```

### (Optional) Save image to file:
```bash
docker save lab1:v1 > my_image.tar
```

KNN model training successful! Accuracy: 92.98%
